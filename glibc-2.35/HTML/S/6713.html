  64       if (normalization_steps != 0)

  71          result is a (N+1)-bit approximation to 1/DIVISOR_LIMB, with the
  72          most significant bit (with weight 2**N) implicit.  */

  74       /* Special case for DIVISOR_LIMB == 100...000.  */
