  56 /* Algorithm:
  57 
  58     x = 2^k z
  59     log2(x) = k + log2(c) + log2(z/c)
  60     log2(z/c) = poly(z/c - 1)

  63 into the ith one, then table entries are computed as
  64 
  65     tab[i].invc = 1/c

  67     tab2[i].chi = (double)c
  68     tab2[i].clo = (double)(c - (double)c)
  69

  71 floating point invc candidates around 1/center and selecting one for which
  72 
  73     1) the rounding error in 0x1.8p10 + logc is 0,
  74     2) the rounding error in z - chi - clo is < 0x1p-64 and
  75     3) the rounding error in (double)log2(c) is minimized (< 0x1p-68).
  76 
  77 Note: 1) ensures that k + logc can be computed without rounding error, 2)
  78 ensures that z/c - 1 can be computed as (z - chi - clo)*invc with close to a
